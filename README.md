# 🐈 Nyx Engine: Sovereign Autonomous Terminal Agent

Nyx is a state-of-the-art, object-driven coding companion running natively inside your terminal. Unlike cloud-dependent tools that charge usage fees, lock you in rigid full-screen boxes, or read your screen as flat text, Nyx uses a completely local mesh to deliver unlimited performance with absolute privacy and zero latency.

## ⚡ The Superpowers

* **The Floating Chat Bar:** An organic, morphing interface that slides smoothly into your active console window without wiping your history.
* **The Ghost Witness:** A silent system observer that intercepts native .NET compiler exceptions in real time to provide instant solutions.
* **A2A Swarm Intelligence:** Seamless communication with peer network nodes via the Agent-to-Agent protocol to synchronize shared solutions.
* **100% Sovereign & Unlimited:** Leverages high-performance local models (like Qwen 2.5 Coder via Ollama) with 0ms latency and zero token costs.
